安装CUDA + NVCC

CUDA Toolkit自带 NVIDIA CUDA 编译器（NVCC）

安装Visual Studio，因为CUDA在安装时，需要VS的里面的工具包来编译。在安装过程中，会自动检测本机是否已经安装了配套的VS版本，如果VS版本和Cuda版本不匹配的话，安装无法进行

下载完安装包后双击，安装选项：工作负载处，勾选“C++的桌面开发（其他的可不勾选，若需要的话，后面可再次安装）
